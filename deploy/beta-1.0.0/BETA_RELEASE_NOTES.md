## Support

**Having issues?**

**App won't launch:**
- Try re-launching from Applications
- Check macOS version (13.0+ required)
- Force quit and relaunch

**Crashes during gameplay:**
- Settings → Send Diagnostics Bundle
- Note what you were doing when it crashed
- Report with diagnostics file

**Notifications not working:**
- Check System Preferences → Notifications
- Ensure Words of the Dead has permission
- Check quiet hours setting in app

**Settings not saving:**
- Quit app completely
- Check ~/Library/Application Support/WordsOfTheDead/ exists
- Restart and try again
