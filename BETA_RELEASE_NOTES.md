# Words of the Dead - Beta 1.0.0

**Release Date**: June 25, 2026  
**Version**: 1.0.0-beta.1  
**Status**: Ready for Beta Testing

---

## What's New

### ✨ Engagement Features
- **Daily Streak Tracking** - Visual streak counter on opening screen
- **📊 Daily Goal Progress** - See your progress toward daily word-learning goal
- **🎯 Rotating Daily Challenges** - New challenges refresh each day
- **🔔 Smart Notifications** - 5 PM reminders (respects quiet hours 9 PM - 9 AM)
- **⚙️ Notification Settings** - Customize when and how you're reminded
- **⏰ Quiet Hours** - No notifications during sleep time

### 🎮 Core Game Features
- Zombie-themed vocabulary learning game
- Progressive difficulty levels (Easy → Hard)
- Track word mastery progression
- Beautiful background images
- Multiple player support
- Session-based gameplay

---

## Installation

1. **Extract the package**
   - Unzip `WordsOfTheDead-beta-1.0.0.zip`

2. **Install the app**
   - Drag "Words of the Dead" to your Applications folder
   - Or double-click to run directly

3. **Launch the app**
   - Open Applications folder
   - Double-click "Words of the Dead"

4. **First Launch**
   - Grant notification permissions when prompted
   - Create your player profile
   - Start playing!

## How to Report Issues

### Option 1: In-App Diagnostics (Recommended)
1. Open the app
2. Click the gear icon (⚙️) in top-right
3. Click "Send Diagnostics Bundle"
4. Email the generated ZIP file

### Option 2: Email with Details
Send to [your-email@example.com]:
- Describe what you were doing
- Steps to reproduce the issue
- Any error messages you see
- Your macOS version (`System Preferences → About`)

### Option 3: App Logs
Manual logs are at: `~/Library/Logs/WordsOfTheDead/`
